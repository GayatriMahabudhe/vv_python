
# Student Management Mini-Project (Modular Python)

This mini-project demonstrates a student management system using multiple Python modules:
- Separate modules: students, marks, attendance, fees
- Report generation module using classes
- Reusable utils module
- Uses standard libs: math, datetime, random
- Import styles: `import`, `from ... import ...`, aliasing
- Module properties: `__name__`, `__file__`, `__dict__`

## Run
```bash
python app_main.py
